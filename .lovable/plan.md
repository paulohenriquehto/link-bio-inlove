

## 📱 InLove Store - Link in Bio

### Visão Geral
Uma página de links elegante no estilo Linktree, usando o design que você criou, com painel administrativo para gerenciar tudo facilmente.

---

### 🎨 Página Pública de Links
- Design exatamente como você criou (cores dourado/bege, animações, visual elegante)
- Logo da InLove Store com selo de verificado
- Botões de links com efeito hover e animação de entrada
- Ícones de redes sociais (Instagram, TikTok, Facebook, Email)
- Responsivo e otimizado para mobile

---

### 🔧 Painel Administrativo
- **Login seguro** para acessar o painel
- **Gerenciar links**: adicionar, editar, remover e reordenar links
- **Destacar link**: marcar um link como "featured" (destaque com animação)
- **Ativar/desativar links** sem precisar deletar

---

### 📊 Analytics de Cliques
- Contagem de cliques em cada link
- Visualizar qual link tem mais acessos
- Dados exibidos no painel de forma simples

---

### 💬 WhatsApp Automático
- Botão que abre conversa no WhatsApp com mensagem pré-configurada
- Você pode personalizar a mensagem no painel admin

---

### 🗄️ Backend com Lovable Cloud
- Banco de dados para armazenar links e contagem de cliques
- Autenticação segura para o painel admin
- Tudo gerenciado automaticamente
